# Run Tests Skill

本スキルは、PRを発行する前にリポジトリ内の全テストを実行し、動作確認および退行（デグレーション）が発生していないかを自動確認するガイドラインです。

## 🛠 テスト実行手順

### 1. テストコマンドの実行
プロジェクト環境に合わせて、以下のコマンドを実行する。

* **Python (uv環境)**:
  ```bash
  uv run pytest
  ```
  ※ テストコードが存在しない場合は、CLIプロトタイプの正常起動確認を行う:
  ```bash
  uv run play-sim --help
  ```

### 2. ログと結果の検証
* すべてのテストが Pass（成功）しているかを確認する。
* エラー（Fail/Error）が発生した場合、スタックトレースを読み取り、PR発行前に原因となったコードを修復する。
* デバッグコードや失敗したアサーションを握りつぶしてテストを無理やり通すことは厳禁とする。
