## 🎯 プロジェクト概要

**PM Simulator** は、プレイヤーがプロジェクトマネージャー（PM）となり、長年にわたりプロジェクトやチームを成功へと導く**通年キャリアマネジメント・シミュレーションゲーム**です。

単発のプロジェクト管理にとどまらず、メンバーの成長や年齢変化、定年退職、新入社員の配属といった中長期的な組織変化と向き合いながら、複数のプロジェクトを継続的に回していきます。

日々の細かいアサインは現場のPL（プロジェクトリーダー）に一任し、PMは限られた行動力（AP）の中でマクロな方針提示、タイムマネジメント、顧客との交渉、チームの不満ケア、リスク予防を行い、プロジェクトの成功率を高めます。

※ 本プロジェクトは現在、Python CLI プロトタイプでのゲームバランス検証を進めており、仕様とロジックが固まり次第、**Godot 4.x (GDScript) による本格的なGUI実装・移植**へと移行する予定です。

## 🎮 ゲームの目的と醍醐味

- **PMの意思決定とタイムマネジメントを体験する**  
  限られた行動ポイント（AP）の中で、どこに時間と労力を集中させるべきかを判断する。
- **トレードオフとジレンマに向き合う**  
  「定例を増やせば状況は見えるがPLの実務が遅れストレスが溜まる」「残業させれば進捗は出るがバグと離職リスクが増す」といったトレードオフの決断。
- **チーム（「人間」）と向き合う組織マネジメント**  
  メンバーの得意・苦手、疲労、年齢（若手の成長・残業耐性 vs ベテランの高品質・ワークライフバランス重視）、隠された本音を解像度の変化を通して理解していく。
- **通年キャリアでの組織育成**  
  頼れるベテランの定年退職を見越し、新人をベテランの元で育成して世代交代を成功させる。

## 🎮 プロトタイプ（CLI版）の実行方法

本プロジェクトでは、本格的なUI実装の前にゲームバランスやルールを検証するための Python CLI プロトタイプを用意しています。
[uv](https://github.com/astral-sh/uv) がインストールされている環境であれば、以下のコマンドで簡単に起動してプレイできます。

```bash
# プロトタイプの起動
uv run play-sim
```
