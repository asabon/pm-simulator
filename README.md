# PM Simulator  
ソフトウェア開発プロジェクトを運営するプロジェクトマネージャー体験シミュレーションゲーム  
（Godot 4 / GDScript / AI支援開発）

---

## 🎯 プロジェクト概要

**PM Simulator** は、プレイヤーがプロジェクトマネージャー（PM）となり、ソフトウェア開発プロジェクトを運営するシミュレーションゲームです。

タスク管理、開発者の状態管理、顧客対応、プロジェクトトラブルの解決、リスク管理、スケジュール調整など、現実の PM が直面する課題をゲームとして体験できます。

本プロジェクトは **Godot 4.x** を使用し、**Google Antigravity による仕様書・コード生成を前提とした開発フロー**を採用します。

---

## 🎮 ゲームの目的

- PM の役割を学ぶ  
- よくあるプロジェクトトラブルへの対処を体験する  
- タスク管理・リスク管理・コミュニケーションの重要性を理解する  
- プロジェクトを成功させて会社の評価を上げる

## 🎮 プロトタイプ（CLI版）の実行方法

本プロジェクトでは、本格的なUI実装の前にゲームバランスを検証するための Python CLI プロトタイプを用意しています。
[uv](https://github.com/astral-sh/uv) がインストールされている環境であれば、以下のコマンドで簡単に起動してプレイできます。

```bash
# プロトタイプの起動
uv run play-sim
```
